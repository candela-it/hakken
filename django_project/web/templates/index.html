What is Hakken?

Service will be used to crowdsource discovery of particular feature types using satellite imagery based on predefined tile like structure.

The discovery process is useful in the preparation/planning phases for projects like HOT Tasking Manager, as it enables to narrow/specialize work area. It could also be useful for general feature discovery tasks. For example, you could setup a task 'discover forested areas in Egypt', so anyone can click through some tasks. Service could also be used to validate finished tasks on the Tasking Manager.
